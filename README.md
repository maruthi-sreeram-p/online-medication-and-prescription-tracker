# Online Medication and Prescription Tracker (Medicare)

A Spring Boot backend system for managing healthcare records, including authentication, patient profiles, and prescription tracking.

## 📖 API Reference & Endpoints
I have generated interactive documentation for the backend controllers. You can use the links below to view the DTO schemas and request/response structures:

* **Main Documentation Portal:** [View All Endpoints](https://suprabath-1209.github.io/online-medication-and-prescription-tracker/DOCUMENTATION/index.html)
* **Auth Controller:** [User Registration & Login](https://suprabath-1209.github.io/online-medication-and-prescription-tracker/DOCUMENTATION/index.html#tag/auth-controller)
* **Patient Controller:** [Patient Records](https://suprabath-1209.github.io/online-medication-and-prescription-tracker/DOCUMENTATION/index.html#tag/patient-controller)
* **Prescription Controller:** [Medication Management](https://suprabath-1209.github.io/online-medication-and-prescription-tracker/DOCUMENTATION/index.html#tag/prescription-controller)

## 🛠 Tech Stack
- **Java:** JDK 22
- **Framework:** Spring Boot 3.2.5
- **Security:** Spring Security with JWT
- **Database:** MySQL 8.0
- **API Spec:** OpenAPI 3.0 / Redoc
